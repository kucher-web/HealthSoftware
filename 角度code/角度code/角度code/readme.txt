1.qv.m是利用转台位置计算方位角俯仰角，第一行输入转台文件名称，跟代码放到统一文件夹内。
转台文件格式是两列数据，第一列为副面轴、第二列为主面轴，中间加一个空格。如需改变馈电1，2，请注意改动安装矩阵代码24行，QV_2_ZH09_Z01_01/QV_1_ZH07_Z01_01。
输出faip,thetap为转台对应的方位角和俯仰角。26-32行计算二维角误差的代码，thetap_real=[];faip_real=[];为对应时刻遥测包里馈电的俯仰角和方位角，error 为二维角误差，这几行不用的话可以注释掉不然代码会报错。
2.back84_1210.m是通过方位角和俯仰角输入转换到84坐标系的代码，d_是最后输出的单位方向矢量。使用时需改变代码里轨道姿态数据
3.wuchafenxi0922.m是计算理论的方位角和俯仰角的代码，使用时需改变代码里轨道姿态数据
